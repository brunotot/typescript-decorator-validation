Function Required

  • Required<T>(options?): FieldDecorator<T>

  • Creates a validator decorator which requires that a value must be present.

    Type Parameters

    • T extends unknown

      The type of the decorated property (any class field).

    Parameters

    • Optional options: DecoratorOptions

      Common decorator options (key, message, groups, etc...)

    Returns FieldDecorator<T>

    A decorator function to use with class fields.

    Key

    Required

    Example

    Example 1: Basic usage

    class Product {
      @Required()
      name: string;
    }

    Example

    Example 2: Supplying a custom error message

    class Product {
      @Required({ message: "Product name is mandatory" })
      name: string;
    }

    Example

    Example 3: Supplying a custom error message and groups

    class Product {
      @Required({
        message: "Product name is mandatory",
        groups: ["CREATE"]
      })
      name: string;
    }
